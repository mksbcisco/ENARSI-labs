# ENARSI-labs
Practice labs/scenario to find the problems with solutions 

You will need GNS3 to work on these labs along with VIRL images from Cisco, details of images used will be mentioned in each lab folder. Each Lab folder will have following content: 
1. List of VIRL Images used in the lab(in readme) 
2. Network Diagram 
3. GNS3 project file
4. A problem statement(in readme)
5. Solution to the problem(as a solution file)
